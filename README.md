# Classificação de Clientes Bancários

## Problema

Bancos e instituições financeiras precisam avaliar o risco de crédito de seus clientes para evitar inadimplência. Identificar se um cliente possui perfil de **bom pagador ou mau pagador** é essencial para decisões de concessão de crédito.

## Objetivo

Desenvolver um modelo de **Machine Learning** capaz de analisar dados históricos de clientes e classificá-los automaticamente como **bons ou maus pagadores**.

## Solução

Este projeto utiliza **Python** para analisar um conjunto de dados de clientes bancários.  
O modelo é treinado com dados históricos e posteriormente testado para verificar sua capacidade de prever corretamente o perfil de pagamento dos clientes.

A análise e o treinamento do modelo são executados em um arquivo **`.ipynb`**, permitindo acompanhar todo o processo de preparação de dados, treinamento e avaliação do modelo.

## Funcionalidades

- Importação e análise de dados de clientes
- Preparação dos dados para treinamento do modelo
- Treinamento de um modelo de classificação
- Teste do modelo para avaliar sua precisão
- Visualização dos resultados diretamente no notebook

## Tecnologias Utilizadas

- Python
- Pandas
- Scikit-learn (sklearn)
- Arquivo `.ipynb` para execução do projeto
